    Om modulet

    Dette modul er måske et af de vigtigste modulet i hele kurset. Det er meget vigtigt at du forstår forskellen på værdibaserede- og referencebaserede variabler, og kan tegne de klassiske og grundlæggende stack- og heap diagrammer i hovedet når du koder. Derfor er dette modul lidt anderledes - der er en del små videoer som forklarer teorien.

    Brug noget tid på dette modul - og især opgaven!

    Stack og heap

    Det klassiske stack/heap diagram kan tegnes som

    På stack’en er findes variabler, og disse indeholder enten rå data (tal, bool, dato, strukturer) eller reference, og stack’en er opdelt i frames. Hver stackframe dækker en metodes virkefelt, og placeres ovenpå hinanden som en stabel tallerkner (derfor kaldes det en stack). Når en metode er færdig fjernes den med LIFO (last in first out) princippet.

    • stack

    • bool

    • byte
    • int
    • long
    • double
    • enum
    • objekter af struct

    • referencer til værdier på heap

    • heap

    • dynamiske data
    • strenge
    • objekter af klasser
    • arrays
    • delegates
    • hvis data ikke er refereret bliver de automatisk fjernet

    Arrays og andre samlinger

    Video Array er en referencebaseret type (5 min.)

    Se video

    Husk at array er en reference baseret type, og det kan jo have en del konsekvenser hvis man eksempelvis sender arrays videre som argument til en metode.

    Find eventuelt den Powerpoint-præsentation der ligger til grund for videoen her.

    Bemærk, at størrelsen på array er underordnet. En referencevariabel vil blot pege på starten af et array på heap’en.

    int[] a = { 5, 1, 7 };
Console.WriteLine($"a = {string.Join(' ', a)}");
int[] b = { 7, 5, 1, 8, 10 };
Console.WriteLine($"b = {string.Join(' ', b)}");
a = b;  // peger på det samme (gamle a array forsvinder)
Console.WriteLine($"a = {string.Join(' ', a)}");
Console.WriteLine($"b = {string.Join(' ', b)}");
a[0] = 99;
Console.WriteLine($"a = {string.Join(' ', a)}");
Console.WriteLine($"b = {string.Join(' ', b)}");
b[1] = 999;
Console.WriteLine($"a = {string.Join(' ', a)}");
Console.WriteLine($"b = {string.Join(' ', b)}");

/*
 ---------- Output: ----------

a = 5 1 7
b = 7 5 1 8 10
a = 7 5 1 8 10
b = 7 5 1 8 10
a = 99 5 1 8 10
b = 99 5 1 8 10
a = 99 999 1 8 10
b = 99 999 1 8 10

*/
