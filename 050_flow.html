    Mønstergenkendelse

    I C# 9 er der nye former for mønstergenkendelse relateret til udtryk. Så istedet for at skrive

    int x = 15;
if(x > 10 && x < 20)
{ }

    kan du benytte nye “and” og “or” kodeord og i stedet skrive

    int x = 15;
if(x is > 10 and < 20)
{ }

    Det kan være lidt nemmere.

    Alternatic til if

    • Brug snippet (switch)
    • break hopper ud
    int i = 10;
switch (i)
{
    case 10:
        Console.WriteLine("i er 10");
        break;

    case 20:
        Console.WriteLine("i er 20");
        break;

    default:
        Console.WriteLine("i er ikke 10 eller 20");
        break;
}


/*
 ---------- Output: ----------

i er 10

*/

    Andet eksempel

    int i = 10;
switch (i)
{
    case 10:
    case 20:
        Console.WriteLine("i er 10 eller 20");
        break;

    default:
        Console.WriteLine("i er ikke 10 eller 20");
        break;
}


/*
 ---------- Output: ----------

i er 10 eller 20

*/

    switch (DateTime.Now.DayOfWeek)

    case DayOfWeek.Saturday:

    Mere avanceret brug af switch

    Switch kan benytte pattern mathing (version 7, 8 og 9):

    Følgende if-struktur

    public static void Vis1(object o)
{
    if (o is Person)
    {
        var p = o as Person;
        Console.WriteLine($"Person hedder {p.Navn}");
    }
    else if (o is Bil)
    {
        var b = o as Bil;
        Console.WriteLine($"Bil er en {b.Mærke}");
    }
    else
    {
        Console.WriteLine($"?");
    }
}

    kan skrives anderledes

    public static void Vis2(object o)
{
    switch (o)
    {
        case Person p:
            Console.WriteLine($"Person hedder {p.Navn}");
            break;
        case Bil b:
            Console.WriteLine($"Bil er en {b.Mærke}");
            break;
        default:
            Console.WriteLine($"?");
            break;
    }
}

    eller mere avanceret ved brug af when:

    public static void Vis3(object o)
{
    switch (o)
    {
        case Person p when string.IsNullOrWhiteSpace(p.Navn):
            Console.WriteLine($"Person har ikke noget navn");
            break;
        case Person p:
            Console.WriteLine($"Person hedder {p.Navn}");
            break;
        case Bil b when b.Mærke != null && b.Mærke.Length <= 2:
            Console.WriteLine($"Bil har et mærkeligt mærke");
            break;
        case Bil b:
            Console.WriteLine($"Bil er en {b.Mærke}");
            break;
        default:
            Console.WriteLine($"?");
            break;
    }
}

    Her er et andet eksempel som kigger på en int:

    int i = 10;
switch (i)
{
    case int n when (n > 0 && n < 10):
        Console.WriteLine($"er n > 0 && n < 10 : {n}");
        break;

    case int n when (n >= 10 && n < 20):
        Console.WriteLine($"er n >= 10 && n < 20 : {n}");
        break;

    case int n when (n >= 20 && n < 30):
        Console.WriteLine($"er n >= 20 && n < 30 :  {n}");
        break;

    default:
        Console.WriteLine("alt andet");
        break;
}

    I C# 9 kan følgende også benyttes:

    int x = 15;
string r = x switch
{
    < 10 => "mindre end 10",
    > 10 and < 20 => "mellem 10 og 20",
    _ => "andet"
};

    Her tildeles r en værdi gennem en lambda med de nye and/or kodeord.

    Do / While

    Video Brug af do/while-løkken (4 min.)

    Se video

    Løber indtil et udtryk er sand. Brug do eller while afhængig af om kode altid skal køre en gang

    while

    DateTime tid = DateTime.Now.AddSeconds(1);
// Løber i 1 sekund
while (DateTime.Now < tid)
{
    // kode
}

    do

    DateTime tid = DateTime.Now.AddSeconds(1);
// Løber i 1 sekund
do
{
    // kode
} while (DateTime.Now < tid);

    • Styring af løkker
    for (int i = 0; i <= 10; i++)
{
    // hvis 2 og 6 - hop til næste iteration
    if (i == 2 || i == 6)
        continue;
    // hvis 8 - hop ud af løkke
    if (i == 8)
        break;

    Console.WriteLine(i);
}

/*
 ---------- Output: ----------

0
1
3
4
5
7

*/

    Goto

    Video Brug af goto (2 min.)

    Se video

    Ind i mellem må man altså gerne bruge goto - men meget mere end et “mærke” i koden gør det uoverskueligt - så pas på med at bruge goto. Men nogen gange gør det koden nemmere at forstå.

    for (int y = 0; y < 10; y++)
{
    for (int x = 0; x < 10; x++)
    {
        if(x==5 && y == 5) {
            goto ud;
        }
    }
}
ud:
    Console.WriteLine("Færdig");

    eller mere kompleks:

    if (DateTime.Now.Millisecond % 2 == 0) {
    for (int i = 0; i < 10; i++)
    {
        if (DateTime.Now.Millisecond > 500)
        {
            goto ud;
        }
        Console.WriteLine(i);
    }
}
else {
    for (int i = 0; i < 5; i++)
    {
        Console.WriteLine(i);
    }
}

ud:
    Console.WriteLine("Færdig");
