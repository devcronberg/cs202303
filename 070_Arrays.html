    Traditionelt array

    Video Traditionelt array (3 min.)

    Se video

    • Tilgængelig i næsten alle sprog
    • Gemmer sammensatte værdier i en datastruktur hvor data kan tilgås gennem et indeks

    • I C# er arrays nulbaseret (index starter fra 0)

    • Vil gerne undgå kode som

    double nedbørMåned1 = 4;
double nedbørMåned2 = 12;
double nedbørMåned3 = 2;
double nedbørMåned4 = 8;
double sum = nedbørMåned1 + nedbørMåned2 + nedbørMåned3 + nedbørMåned4;
Console.WriteLine($"Sum {sum:N2}");
double gns = sum / 4;
Console.WriteLine($"Gennemsnit {gns:N2}");

/*
 ---------- Output: ----------

Sum 26,00
Gennemsnit 6,50

*/
    • men i stedet bruge arrays for det er meget nemmere at arbejde med
    double[] nedbør = { 4, 12, 2, 8 };
double sum = 0;
for (int i = 0; i < nedbør.Length; i++)
    sum += nedbør[i];
Console.WriteLine($"Sum {sum:N2}");
double gns = sum / nedbør.Length;
Console.WriteLine($"Gennemsnit {gns:N2}");

/*
 ---------- Output: ----------

Sum 26,00
Gennemsnit 6,50

*/

    int[] a = new int[] { 10, 5, 1, 7, 1, 6 };
Console.WriteLine(string.Join(' ', a));

// Kopi af array til et nyt på heap

Console.WriteLine();

b[0] = 20;

// Kopiering af værdier
int[] c = new int[10];
a.CopyTo(c, 0);
Console.WriteLine();
Console.WriteLine(string.Join(' ', c));

int[] d = new int[10];
a.CopyTo(d, 4);
Console.WriteLine();
Console.WriteLine(string.Join(' ', d));

Length: 6

10 5 1 7 1 6

10 5 1 7 1 6 0 0 0 0

    int[] a = new int[] { 10, 5, 1, 7, 1, 6 };
Console.WriteLine(string.Join(' ', a));

System.Array.Resize(ref a, 10);

    Hvad med metoder?

    • Igen - arrays er en reference baseret variabel så hvis du sender en sådan med til en metode så kan du foretage rettelser i værdierne. Det er jo ikke sikkert at den der kalder din metode er glad for det.
    int[] a = { 5, 1, 7 };
Console.WriteLine($"a = {string.Join(' ', a)}");
Console.WriteLine($"Mindste værdi: { FindMindsteVærdi_FEJL(a)}");
Console.WriteLine($"a = {string.Join(' ', a)}");

Console.WriteLine();
int[] b = { 5, 1, 7 };
Console.WriteLine($"b = {string.Join(' ', b)}");
Console.WriteLine($"Mindste værdi: { FindMindsteVærdi(b)}");
Console.WriteLine($"b = {string.Join(' ', b)}");

int FindMindsteVærdi_FEJL(int[] array)
{
    // Sorterer det oprindelige array!!!
    System.Array.Sort(array);
    return array[0];
}

int FindMindsteVærdi(int[] array)
{
    int[] kopi = array.Clone() as int[];
    System.Array.Sort(kopi);
    return kopi[0];
}


/*
 ---------- Output: ----------

a = 5 1 7
Mindste værdi: 1
a = 1 5 7

b = 5 1 7
Mindste værdi: 1
b = 5 1 7

*/

    Flere dimensioner

    • Kan arbejde med mange dimensioner
    string[,] skakBræt = new string[8, 8];
skakBræt[0, 0] = "Ts";
skakBræt[0, 1] = "Hs";
skakBræt[0, 2] = "Ls";
skakBræt[0, 3] = "Ds";
skakBræt[0, 4] = "Ks";
skakBræt[0, 5] = "Ls";
skakBræt[0, 6] = "Hs";
skakBræt[0, 7] = "Ts";

for (int i = 0; i < 8; i++)
{
    skakBræt[1, i] = "Bs";
    skakBræt[6, i] = "Bh";
    skakBræt[7, i] = skakBræt[0, i].Replace("s", "h");
}

for (int række = 0; række < 8; række++)
{
    for (int kolonne = 0; kolonne < 8; kolonne++)
    {
        Console.Write(skakBræt[række, kolonne] + " ");
    }
    Console.WriteLine();
}


/*
 ---------- Output: ----------

Ts Hs Ls Ds Ks Ls Hs Ts
Bs Bs Bs Bs Bs Bs Bs Bs




Bh Bh Bh Bh Bh Bh Bh Bh
Th Hh Lh Dh Kh Lh Hh Th

*/

    En konsol applikation med parametre

    Hvis en konsol applikation startes med parametre kan det fanges i args-arrayet:

    test.exe a b c

    eller

    dotonet test a b c

    Parametre kan findes i args:

    using System;

namespace DemoArgs
{
    class Program
    {
        static void Main(string[] args)
        {
            if (args != null)
                for (int i = 0; i < args.Length; i++)
                    Console.WriteLine($"{i} = {args[i]}");
        }
    }
}

    PS argumenter kan også sættes under Debug-fanen i egenskaberne til projektet (højreklik på projektet). PS brug eventuelt en af de mange NuGet-pakker for at gøre tilgang til argumenter nemmere - CommandLineArgumentsParser

    Avancerede metoder

    Der findes en del statiske metoder på System.Array, som egner sig bedst til brug af lambda. Flere vælge dog at bruge LINQ men metoderne er tilgængelige.

    int[] a = { 5, 4, 2, 1, 3, 6, 8, 12, 6, 3, 1 };

// Skift type
byte[] b = Array.ConvertAll<int, byte>(a, (x) => (byte)x);

// Gennemløb
Array.ForEach(a, x => Console.WriteLine(x));

// Findes element
bool c = Array.Exists(a, x => x == 1);      // true

// Find
int d = Array.Find(a, x => x == 12);        // 7

// Find alle
int[] e = Array.FindAll(a, x => x < 5);     // 4, 2, 1, 3, 3, 1

// Er det sandt at alle elementer er ...
bool f = Array.TrueForAll(a, x => x < 20);  // true

    System.Index og System.Range

    Den lidt specielle ^ og .. syntaks der kendes fra andre sprog kan også bruges i C#

    • ^ betyder fra enden
      • System.Index
      • ^0 returnerer længden
    • .. betyder range
      • System.Range
    int[] nr = new[] { 5, 7, 1, 3, 9, 10, 0, 1, 2, 7 };

VisArray(nr[..]);           // 5 7 1 3 9 10 0 1 2 7
VisArray(nr[..4]);          // 5 7 1 3
VisArray(nr[4..]);          // 9 10 0 1 2 7
VisArray(nr[1..5]);         // 7 1 3 9


// Bagfra
Console.WriteLine(nr[^1]);  // 7
Console.WriteLine(nr[^5]);  // 10

VisArray(nr[^2..]);         // 2 7
VisArray(nr[^5..^3]);       // 10 0

System.Index fra = ^4;
System.Index til = ^2;
VisArray(nr[fra..til]);     // 0 1

System.Range r = 1..4;
VisArray(nr[r]);            // 7 1 3

void VisArray(int[] a)
{
    Console.WriteLine(string.Join(' ', a));
}

    Brug af params

    • Brug evt params i argument listen til metoder
    • Argumenter overføres som array
    Test1(1);
Test1(1, 2);
Test1(1, 2, 3);
Test2(1, 1);
Test2(1, 1, 2);
Test2(1, 1, 2, 3);

void Test1(params int[] args)
{
    Console.WriteLine($"Kaldt med {String.Join(", ", args)}");
}

void Test2(int a, params int[] args)
{
    Console.WriteLine($"Kaldt med a={a} og {String.Join(", ", args)}");
}




/*
 ---------- Output: ----------

Kaldt med 1
Kaldt med 1, 2
Kaldt med 1, 2, 3
Kaldt med a=1 og 1
Kaldt med a=1 og 1, 2
Kaldt med a=1 og 1, 2, 3

*/
