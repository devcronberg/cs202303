    Kommunikation med XML

    Om modulet

    Dette modul vil introducere dig til brugen af XML på forskellige måder i C#.

    System.XML

    Brug af den “gamle” System.Xml.Document klasse

    Helt grundlæggende brug:

    • Load (loader fil)
    • LoadXml (loader string)
    • DocumentElement (rod)
    • SelectSingleNode (returnerer en enkelt node via XPath)
    • SelectNodes (returnerer flere noder via XPath)

    En enkelt node kan være repræsenteret med XmlNode som beatår af

    • InnerText
    • Attributtes
    • SelectSingleNode (returnerer en enkelt node via XPath)
    • SelectNodes (returnerer flere noder via XPath)

    Grundlæggende XPath syntax

    //title[@lang]  
Selects all the title elements that have an attribute named lang

//title[@lang='en'] 
Selects all the title elements that have a "lang" attribute with a value of "en"

/bookstore/book[1]  
Selects the first book element that is the child of the bookstore element

/bookstore/book[last()] 
Selects the last book element that is the child of the bookstore element

/bookstore/book[last()-1]
Selects the last but one book element that is the child of the bookstore element

/bookstore/book[position()<3]   
Selects the first two book elements that are children of the bookstore element

/bookstore/book[price>35.00]    
Selects all the book elements of the bookstore element that have a price element with a value greater than 35.00

/bookstore/book[price>35.00]/title  
Selects all the title elements of the book elements of the bookstore element that have a price element with a value greater than 35.00

    Load data

    • Brug Load for at hente XML til en samling af XElement
    • Brug Where, Order, Group til at skabe et udtryk
    // kræver using System.Xml.Linq og using System.Linq;

// Hele dokumentet
XDocument doc = XDocument.Load(@"data.xml");
foreach (var item in doc.Elements())
{
    Console.WriteLine(item);
}

Console.WriteLine();

// Del af dokumentet
IEnumerable<XElement> personer = XElement.Load(@"data.xml").Elements();
foreach (var item in personer)
{
    Console.WriteLine(item);
}

    Læs data

    • Brug element Value til at læse data
    // kræver using System.Xml.Linq og using System.Linq;

// Hele dokumentet
var doc = XElement.Load(@"data.xml").Descendants("person");

foreach (var item in doc)
    Console.WriteLine(item.Element("navn").Value);

/*
 ---------- Output: ----------

Mikkel
Mathias

    Find data

    • Brug where og order mv til at finde data
    // kræver using System.Xml.Linq og using System.Linq;
XElement personer = XElement.Load(@"data.xml");
Console.WriteLine(personer);

var res1 = from node in personer.Descendants("person")
           where (int)node.Element("alder") > 13 orderby (int)node.Element("alder") descending select node;
var res2 = personer.Descendants("person").Where(i => (int)i.Element("alder") > 13)
    .OrderByDescending(i => (int)i.Element("alder")); ;

Console.WriteLine();
foreach (var item in res1)
{
    Console.WriteLine(item.Element("navn").Value);
}
Console.WriteLine();
foreach (var item in res2)
{             
    Console.WriteLine(item.Element("navn").Value);

XDocument xDoc = new XDocument(
new XDeclaration("1.0", "UTF-16", null),
new XElement("Personer",
    new XElement("Person",
        new XComment("Kommentar"),
        new XElement("Id", "1"),
        new XElement("Navn", "Mathias")                    
        )));

Console.WriteLine(xDoc);

 ---------- Output: ----------

<Personer>
  <Person>

    <Id>1</Id>

    Manipulering

    • Opdater, slet, tilføj
    // kræver using System.Xml.Linq og using System.Linq;
XElement personer = XElement.Load(@"data.xml");
Console.WriteLine(personer);
Console.WriteLine();
var res1 = from node in personer.Descendants("person")
           where (int)node.Attribute("id") == 1
           select node;
res1.FirstOrDefault().Element("navn").Value = "MIKKEL";


personer.Descendants("person").
    Where(i => (int)i.Attribute("id") == 2)
    .FirstOrDefault().Element("navn").Value = "MATHIAS";


var ny = new XElement("Land", "Danmark", new XAttribute("eu","true"));

personer.Descendants("person").
    Where(i => (int)i.Attribute("id") == 2)
    .FirstOrDefault().Add(ny);

personer.Descendants("person").
    Where(i => (int)i.Attribute("id") == 3)
    .FirstOrDefault().Element("alder").Remove();


Console.WriteLine(personer);
